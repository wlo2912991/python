### list資料型態

```
很像陣列(array)的list(清單|列表)
x = [11, 22, 33,44,55]

但list可以儲存不同資料型態的元素
x = [6, "Dragon", [10, 21, 32]]
```
### List 的indices(索引)
```
 若x = ["first", "second", "third", "fourth"]
 則下列分別為何?哪一個會出錯?
 x[1]
 x[3]
 x[5]
 x[-1]
 x[-2]

 x[1:3]
 x[:3]
 x[1:-1]
 x[:]
```
### 修改列表
```
>>> x = [1, 2, 3, 4]
>>> x[1] = "two"
>>> x
[1, 'two', 3, 4]
```
