## 複數
```
python用 nj 表示虛數
n為任意整數或浮點數
j值= -1 平方根
python 會用括弧顯示結果

可以用 real、imag 分別取得實部以及虛部
```
```
>>> 3 + 2j - (4 + 4j)
(-1-2j)  <--------------用括弧表示-1-2j是單一物個件
>>> (1 + 2j) * (3 + 4j)
(-5+10j)
>>> 1j*1j
(-1+0j)
```
```
>>> z = 3+5j
>>> z.real
3.0
>>> z.imag
5.0
```
