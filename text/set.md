# set 集合
```
set跟list、tuple不同在於
set沒有順序
且重複的資料會自動刪除

x = [11, 22, 33,44,55]   
這是list型態

x = (11, 22, 33,44,55)
這是tuple型態

x = [11, 22, 33,44,55]
這是set型態

可以用set將其他序列轉為set型態
```

## and or xor
```
可以用&來求出兩個set的交集 相當於邏輯中的AND
#可以用|來求出兩個set的聯集 相當於邏輯中的OR
可以用|來求出兩個set的對稱差(只屬於一個set而不屬於另外一個) 相當於邏輯中XOR
```
```
>>> x = set([1, 2, 3, 6])
>>> y = set([1, 7, 8, 9])
>>> x | y                 #可以用|來求出兩個set的聯集 相當於邏輯中的OR
{1, 2, 3, 6, 7, 8, 9}
>>> x & y                  #可以用&來求出兩個set的交集 相當於邏輯中的AND
{1}
>>> x ^ y              #可以用|來求出兩個set的對稱差(只屬於一個set而不屬於另外一個) 相當於邏輯中XOR
{2, 3, 6, 7, 8, 9}
```
