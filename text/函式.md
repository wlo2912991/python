# 函式
```
函式格式

def(參數 1, 參數 2 , ....):
    ......
    程式區塊
    ......

參數不是必要的
```
```
>>> def fact(n): 
...     """回傳n的階乘"""            
...     r = 1
...     while n > 0:
...         r = r * n
...         n = n - 1
...     return r                                     
...


>>> fact(3, 3)
27

```

## 用參數名稱傳遞引數


```
>>> def power(x, y=2):
...     r = 1
...     while y > 0:
...         r = r * x
...         y = y - 1
...     return r

>>> power(2, 3)
8
>>> power(3, 2)
9
>>> power(y=2, x=3)   <----- 不照順序傳遞 照名稱
9
```
